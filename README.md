# cneo programming language
Simple, secure, fast and compiled language for developing maintainable, scalable, reliable, and efficient software. Compiles itself with zero library dependencies and transpiled to C.

* Safety: no null, no globals, no undefined behavior (wip), immutability by default
* Simplicity: the language can be learned over the course of a weekend
* Fast compilation
* Easy to develop: cneo compiles itself in less than a second
* Performance: as fast as C (CNeo's main backend compiles to human-readable C)
* Hot code reloading
* Easy cross-compilation
* Built-in ORM
* etc etc
